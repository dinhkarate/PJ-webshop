## :anchor: Bắt đầu làm việc

1. Clone repository

```console
git clone https://github.com/SVUIT/mmtt.git
```

2. Tạo branch mới

```console
git branch <branch-name>
```
**Lưu ý:** Format tên branch `TenHo`

3. Chuyển sang branch vừa tạo và làm việc trên branch đấy

```console
git checkout <branch-name>
```
